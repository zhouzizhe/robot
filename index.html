1. You can use this toy to draw grid no greater than 100 in width and length, although I do not recommend drawing a too large grid as it will burden your browser with calculations.
2. You can give input and get the output from the robot.

4. The toy also supports autoplay and adjusts the playing speed to help you enjoy the robot's path.
5. You should note that some of the settings of this toy are different from those for the submission. For example, in this toy, the robot can take no more than 500,000 steps and doesn't check to see if the same memory is printed more than once.
